## 📌 Overview

A full-stack role-based Islamic learning platform designed to manage interactions between students, teachers, and administrators in a structured educational environment.

The system implements secure authentication, user approval workflows, and role-based access control (RBAC), ensuring that each user interacts only with their assigned data and responsibilities.

---

## 🧠 System Architecture

The platform follows a role-based architecture:

- **Admin**: Manages users, approvals, and system operations
- **Teacher**: Uploads materials, assignments, and manages students
- **Student**: Accesses assigned content and submits work

This design ensures data isolation, security, and scalability.
